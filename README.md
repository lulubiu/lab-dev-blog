## 项目简介

这是一个简洁的静态网站，用来展示我做的各种小工具和创意 idea。每个工具都是独立的页面，你可以直接在浏览器中使用。

### 特点

- 🎨 **简洁美观** - 采用渐变背景和玻璃拟态设计
- ⚡ **即开即用** - 纯前端实现，无需后端服务
- 📱 **响应式设计** - 完美适配手机、平板、电脑
- 🎯 **轻量快速** - 使用 Tailwind CSS，加载快速

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript
- 无框架，纯静态网站

## 本地运行

这是一个纯静态网站，不需要任何构建工具或服务器。

### 方法 1：直接打开

直接用浏览器打开 `index.html` 文件即可。

### 方法 2：使用本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`
